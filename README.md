# Shivin Bassi - Portfolio Website

A premium, futuristic personal portfolio website showcasing ML Engineering, Full Stack Development, and Data Analytics projects.

## 🚀 Features

- **Modern Design**: Dark theme with glassmorphism, gradient effects, and smooth animations
- **Fully Responsive**: Optimized for mobile, tablet, and desktop devices
- **Interactive UI**: Smooth scroll animations, hover effects, and particle background
- **Premium Components**:
  - Animated hero section with particle background
  - Skills showcase with animated cards
  - Project gallery with detailed information
  - Experience timeline
  - Certifications grid
  - Contact form with glassmorphism design
  - Custom cursor glow effect
  - Scroll progress indicator

## 🛠️ Tech Stack

- **React** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Vite** - Build tool
- **Lucide React** - Icons

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎨 Design Features

- **Glassmorphism**: Modern frosted glass effect on cards
- **Gradient Animations**: Dynamic color transitions
- **Particle Background**: Interactive animated particles in hero section
- **Custom Cursor Glow**: Radial gradient that follows the cursor
- **Scroll Progress**: Visual indicator of page scroll progress
- **Floating Blobs**: Animated gradient blobs in the background
- **Smooth Transitions**: Page and component animations with Framer Motion
